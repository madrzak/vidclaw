Changelog

All notable updates to VidClaw, the self-hosted AI dashboard for OpenClaw agent management.

v1.5.0 New

February 2025
  • Added credentials manager for secure secret storage
  • New "Delete All Done" bulk action for the Kanban board
  • Improved task card display for completed items
  • Better memory management panel

v1.4.0 Improved

January 2025
  • SEO improvements — optimized robots.txt and sitemap.xml
  • Optimized README for discoverability
  • Cross-platform setup script (Linux + macOS)

v1.3.0 New

December 2024
  • AI cost tracking panel with per-model breakdowns
  • Rate limit progress bars matching Anthropic's windows
  • Model switching from the navbar
  • Daily and monthly spending summaries

v1.2.0 New

November 2024
  • Skills Manager — browse, enable, disable, and create skills
  • Soul Editor with version history and starter templates
  • Content browser with markdown preview

v1.1.0 Improved

October 2024
  • Activity calendar view parsed from memory files
  • Task priority levels (P1–P4)
  • Drag-and-drop improvements on mobile

v1.0.0

September 2024
  • Initial release of VidClaw
  • Kanban task board with four columns
  • Basic usage tracking
  • SSH tunnel security model
  • One-command setup via setup.sh

For the full commit history, see the GitHub commits. Explore all features or read the setup guide.
